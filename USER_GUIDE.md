## 1. 프로그램 소개

본 프로그램은 월 1회 수행하는 원천징수 관련 업무를 자동으로 처리해 주는 도구입니다.

**현재 사용 가능한 기능 (3가지):**

| 단계 | 기능 | 설명 |
|------|------|------|
| 1단계 | 수임처 목록 조회 | WEHAGO에서 담당 수임처 목록을 자동으로 가져옵니다 |
| 2단계 | 국민건강보험 | 수임처별 결정내역을 PDF, Excel 파일로 자동 다운로드합니다 |
| 3단계 | 국민연금 | 수임처별 결정내역을 PDF, Excel 파일로 자동 다운로드합니다 |

**향후 추가 예정 기능:** 급여자료입력, 원천이행상황신고서, 원천전자신고, 홈택스 신고 (순차 개발 중)

---

## 2. 시작하기 전 준비사항

아래 두 가지가 미리 준비되어 있어야 프로그램을 사용할 수 있습니다.

**필수 항목:**

1. **Google Chrome 웹브라우저 설치**
   - PC에 Chrome이 설치되어 있어야 합니다
   - 설치 경로는 기본 경로(C:\Program Files\Google\Chrome)여야 합니다

2. **공동인증서 로그인 가능**
   - WEHAGO, 국민건강보험 EDI, 국민연금 EDI 각 포털에 공동인증서로 로그인할 수 있어야 합니다
   - 본 프로그램은 로그인을 대신하지 않습니다. 사용자가 직접 로그인해야 합니다

3. **Python 프로그램 설치** (exe 파일로 실행하는 경우 불필요)
   - Python 3.10 이상이 설치되어 있어야 합니다

---

## 3. 최초 설치 방법

프로그램을 처음 사용할 때만 아래 절차를 1회 수행합니다.

1. 프로그램 폴더 안에 있는 **setup.bat** 파일을 찾습니다
2. 해당 파일 위에서 **마우스 오른쪽 버튼**을 누릅니다
3. **"관리자 권한으로 실행"**을 선택합니다
4. 검은 창이 열리면서 설치가 진행됩니다
5. "설치가 완료되었습니다" 메시지가 나오면 아무 키나 눌러 창을 닫습니다

> 주의: setup.bat은 최초 1회만 실행하면 됩니다. 이후에는 실행할 필요가 없습니다.

## 4. 프로그램 실행 방법

**방법 A: 실행 파일로 시작 (권장)**

1. 프로그램 폴더 안에 있는 **원천징수자동화.exe** 파일을 더블클릭합니다
2. 프로그램 화면이 열립니다

**방법 B: 명령어로 시작**

1. 프로그램 폴더에서 주소 표시줄에 `cmd`를 입력하고 Enter를 누릅니다
2. 검은 창에 `python gui_main.py`를 입력하고 Enter를 누릅니다
3. 프로그램 화면이 열립니다

## 5. 화면 구성 안내

프로그램을 실행하면 다음과 같은 화면이 나타납니다.

```
+---------------------------------------------------------------+
|  [연도 선택] [월 선택]     [시작] [일시정지] [정지]           |
+---------------+-----------------------------------------------+
|               |                                               |
|  1 수임처     |                                               |
|    리스트     |         수임처 목록 테이블                     |
|               |       (수임처명, 사업자번호 등)                |
|  2 건강보험   |                                               |
|               |                                               |
|  3 국민연금   |                                               |
|               +-----------------------------------------------+
|  4 (개발중)   |                                               |
|  5 (개발중)   |         진행 상황 로그                         |
|  6 (개발중)   |       (현재 처리 중인 내용 실시간 표시)        |
|  7 (개발중)   |                                               |
+---------------+-----------------------------------------------+
```

- **왼쪽:** 7개 단계 버튼 (현재 1~3단계 사용 가능)
- **오른쪽 위:** 수임처 목록 테이블
- **오른쪽 아래:** 진행 상황을 실시간으로 보여주는 로그 화면
- **상단:** 연도/월 선택, 시작/일시정지/정지 버튼

---

## 6. 1단계: 수임처 목록 조회

WEHAGO에서 담당 수임처 목록을 자동으로 가져옵니다. 다른 모든 단계를 진행하기 전에 먼저 수행해야 합니다.

**절차:**

1. 왼쪽 메뉴에서 **"1 수임처리스트"** 버튼을 클릭합니다
2. 오른쪽 화면에 **"새로 가져오기"** 버튼이 나타납니다
3. **"새로 가져오기"** 버튼을 클릭합니다
4. Chrome 브라우저가 자동으로 열리며 WEHAGO 로그인 화면이 나타납니다
5. **공동인증서로 직접 로그인**합니다
6. 로그인이 완료되면 수임처 목록이 자동으로 수집됩니다
7. 수집이 완료되면 오른쪽 테이블에 수임처 목록이 표시됩니다

> 안내: 수임처 목록은 프로그램에 자동 저장됩니다. 프로그램을 껐다 켜도 목록이 유지됩니다.

---

## 7. 2단계: 국민건강보험 결정내역 다운로드

수임처별로 국민건강보험 결정내역을 PDF, Excel 파일로 자동 다운로드합니다.

**절차:**

1. 화면 상단에서 **조회할 연도와 월**을 선택합니다 (예: 2026년 4월)
2. 왼쪽 메뉴에서 **"2 건강보험"** 버튼을 클릭합니다
3. 화면 상단의 **"시작"** 버튼을 클릭합니다
4. Chrome 브라우저가 자동으로 열리며 국민건강보험 EDI 로그인 화면이 나타납니다
5. **공동인증서로 직접 로그인**합니다
6. 로그인이 완료되면 수임처별로 자동 처리가 시작됩니다
7. 오른쪽 테이블에서 각 수임처의 진행 상태를 실시간으로 확인할 수 있습니다
8. 하단 로그 화면에서 상세 진행 내용을 확인할 수 있습니다

**저장 위치:** 바탕화면에 **"수임처명_국민건강보험"** 폴더가 자동 생성되며, 그 안에 PDF와 Excel 파일이 저장됩니다.

---

## 8. 3단계: 국민연금 결정내역 다운로드

수임처별로 국민연금 결정내역을 PDF, Excel 파일로 자동 다운로드합니다. 진행 방식은 2단계와 동일합니다.

**절차:**

1. 화면 상단에서 **조회할 연도와 월**을 선택합니다
2. 왼쪽 메뉴에서 **"3 국민연금"** 버튼을 클릭합니다
3. 화면 상단의 **"시작"** 버튼을 클릭합니다
4. Chrome 브라우저가 자동으로 열리며 국민연금 EDI 로그인 화면이 나타납니다
5. **공동인증서로 직접 로그인**합니다
6. 로그인이 완료되면 수임처별로 자동 처리가 시작됩니다
7. 진행 상황은 오른쪽 테이블과 하단 로그에서 실시간 확인합니다

**저장 위치:** 바탕화면에 **"수임처명_국민연금"** 폴더가 자동 생성되며, 그 안에 PDF와 Excel 파일이 저장됩니다.

## 9. 자주 묻는 질문 (FAQ)

---

**Q: "수임처 조회 실패: no such table: steps" 에러 메시지가 나타납니다**

A: 최초 설치 과정이 누락된 것입니다. setup.bat 파일을 마우스 오른쪽 버튼으로 눌러 **"관리자 권한으로 실행"**을 선택하여 한 번만 실행하시면 해결됩니다.

---

**Q: Chrome 브라우저가 열리지 않습니다**

A: Google Chrome이 설치되어 있는지 확인하세요. Chrome이 기본 설치 경로가 아닌 다른 곳에 설치된 경우 정상 작동하지 않을 수 있습니다.

---

**Q: 로그인 화면에서 프로그램이 멈춰 있는 것 같습니다**

A: 정상입니다. 프로그램은 사용자가 직접 로그인할 때까지 최대 15분간 기다립니다. 공동인증서로 로그인하시면 자동으로 다음 단계로 진행됩니다.

---

**Q: 처리 중간에 에러가 발생해서 멈췄습니다**

A: **"시작"** 버튼을 다시 누르면 이미 완료된 수임처는 자동으로 건너뛰고, 실패한 수임처부터 이어서 진행합니다.

---

**Q: 수임처 목록에 누락된 곳이 있습니다**

A: 1단계에서 **"새로 가져오기"** 버튼을 다시 클릭하면 WEHAGO에서 최신 수임처 목록을 다시 가져옵니다.

---

**Q: 프로그램을 중간에 중단하고 싶습니다**

A: 화면 상단의 **"정지"** 버튼을 누르면 안전하게 중단됩니다. 다음에 다시 실행하면 이어서 진행할 수 있습니다.

---

**Q: 프로그램을 일시정지하고 싶습니다**

A: 화면 상단의 **"일시정지"** 버튼을 누르면 잠시 멈춥니다. 다시 **"시작"** 버튼을 누르면 계속 진행됩니다.

---

**Q: 다운로드된 파일은 어디에서 확인하나요**

A: 바탕화면에 수임처별 폴더가 자동으로 생성됩니다.
- 국민건강보험: 바탕화면 > "수임처명_국민건강보험" 폴더
- 국민연금: 바탕화면 > "수임처명_국민연금" 폴더
