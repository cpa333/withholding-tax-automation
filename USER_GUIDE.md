## 1. 프로그램 소개

본 프로그램은 월 1회 수행하는 원천징수 관련 업무를 자동으로 처리해 주는 도구입니다.

**현재 사용 가능한 기능 (8가지):**

| 단계 | 기능 | 설명 |
|------|------|------|
| 1단계 | 수임처 목록 조회 | WEHAGO에서 담당 수임처 목록을 자동으로 가져옵니다 |
| 2단계 | 국민건강보험 | 수임처별 결정내역을 PDF, Excel 파일로 자동 다운로드합니다 |
| 3단계 | 국민연금 | 수임처별 결정내역을 PDF, Excel 파일로 자동 다운로드합니다 |
| 4단계 | 급여자료입력 | 건강보험·국민연금 등 원천데이터를 WEHAGO 엑셀에 반영하여 업로드합니다 |
| 5단계 | 급여명세 PDF | 수임처별 급여명세서를 PDF로 자동 발급합니다 |
| 6단계 | 원천이행상황신고서 | WEHAGO에서 원천징수이행상황신고서 마감 처리를 자동 수행합니다 |
| 7단계 | 원천전자신고 | WEHAGO에서 원천징수전자신고 파일을 수임처별로 자동 제작합니다 |
| 8단계 | 홈택스 원천세 신고 | 7단계에서 생성된 신고파일을 홈택스에 업로드하여 검증합니다 |

## 2. 시작하기 전 준비사항

아래 항목이 미리 준비되어 있어야 프로그램을 사용할 수 있습니다.

**필수 항목:**

1. **Google Chrome 웹브라우저 설치**
   - [공식 사이트(google.com/chrome)](https://www.google.com/chrome/)에서 **stable 버전**을 설치하세요
   - 설치 경로는 일반/시스템 설치 모두 자동 인식합니다 (기본 경로 고정 불필요)
   - **설치 후 한 번이라도 실행**해 프로필이 생성되어 있어야 합니다
   - 주의: 네이버 웨일·엣지·브레이브 등 "크로미움 기반" 브라우저는 인식하지 못합니다. Google Chrome이어야 합니다

2. **프로그램 계정 (이메일/비밀번호)**
   - 첫 실행 시 로그인이 필요합니다. **계정은 담당자가 사전 발급**합니다
   - 직접 회원가입은 불가능합니다. 계정이 없으면 담당자에게 문의하세요
   - 첫 로그인에는 **인터넷 연결**이 필요합니다 (이후 마지막 로그인으로부터 3일까지는 오프라인 사용 가능)

3. **공동인증서 로그인 가능**
   - WEHAGO, 국민건강보험 EDI, 국민연금 EDI, 홈택스 각 포털에 공동인증서/간편인증으로 로그인할 수 있어야 합니다
   - 본 프로그램은 포털 로그인을 대신하지 않습니다. 사용자가 직접 로그인해야 합니다

4. **WehagoNTS 프로그램 설치** (7단계 원천전자신고 사용 시)
   - WEHAGO에서 제공하는 전자신고 파일 제작 프로그램(WehagoNTS)이 설치되어 있어야 합니다
   - 7단계 실행 시 자동으로 실행되며, 설치되지 않은 경우 파일 저장 단계에서 오류가 발생합니다

> 참고: 설치형(.exe)으로 사용할 때는 **Python이 필요 없습니다.** 프로그램에 필요한 모든 모듈이 설치 파일에 함께 포함되어 있습니다.

## 3. 최초 설치 방법

담당자가 알려준 설치 파일(`whta_setup.exe` 또는 `원천징수자동화_설치.exe`)을 받아 아래 절차를 1회 수행합니다. 관리자 권한은 필요하지 않습니다.

**1. 설치 파일 실행**

1. 받은 설치 파일을 더블클릭합니다

**2. 파란 화면이 뜨면 (Windows SmartScreen — 정상입니다)**

인터넷에서 받은 파일이라 Windows가 한 번 더 확인합니다. **바이러스가 아니니 안심하세요.**

1. 화면 아래 작은 글씨 **"추가 정보"** 를 클릭합니다
2. 나타난 **"실행"** 버튼(회색)을 클릭합니다

> 💡 USB나 사내 공유 폴더로 직접 받은 경우에는 이 파란 화면이 뜨지 않습니다.

**3. 설치 마법사 진행**

1. **[다음]** → **[설치]** → **[완료]** 순서로 클릭합니다
2. 바탕화면과 시작 메뉴에 **"원천징수 자동화"** 바로가기가 생깁니다

> ⚠️ **Google Chrome이 설치되어 있지 않으면 설치가 진행되지 않습니다.**
> 안내 창이 뜨면 먼저 [google.com/chrome](https://www.google.com/chrome/)에서 Chrome을 설치하고, 설치 파일을 다시 실행하세요.

> 주의: 설치는 최초 1회만 하면 됩니다. 이후에는 **4. 프로그램 실행 방법** 만 따르면 됩니다.

---

## 4. 프로그램 실행 방법

1. 바탕화면(또는 시작 메뉴)의 **"원천징수 자동화"** 아이콘을 더블클릭합니다

2. **로그인 창**이 나타나면 담당자에게 받은 **이메일/비밀번호**로 로그인합니다
   - 최초 1회 로그인 후에는 자동 로그인되어 다시 물어보지 않습니다
   - 인터넷이 잠시 끊겨도 마지막 로그인으로부터 **3일** 이내면 계속 사용할 수 있습니다

3. 로그인에 성공하면 메인 화면이 열립니다

> 계정이 없으면 담당자에게 문의하세요. (로그인 창 하단에도 안내가 표시됩니다)

## 5. 화면 구성 안내

프로그램을 실행하면 다음과 같은 화면이 나타납니다.

```
+---------------------------------------------------------------+
|  [연도 선택] [월 선택]  [담당자: 이름입력]  [일시정지]        |
|                        ↑ 1단계 선택 시 표시                     |
+---------------+-----------------------------------------------+
|               |  [새로가져오기]  [모두삭제]                     |
|  1 수임처     |                                               |
|    리스트     |         수임처 목록 테이블                     |
|               |       (수임처명, 사업자번호 등)                |
|  2 건강보험   |                                               |
|               |                                               |
|  3 국민연금   |                                               |
|               |                                               |
|  4 급여자료   |                                               |
|    입력       |                                               |
|  5 급여명세   |                                               |
|    PDF        |                                               |
|  6 원천이행   |                                               |
|    신고서     +-----------------------------------------------+
|               |         진행 상황 로그                         |
|  7 전자신고   |       (현재 처리 중인 내용 실시간 표시)        |
|               |                                               |
|  8 홈택스     |                                               |
|    신고       |                                               |
+---------------+-----------------------------------------------+
```

- **왼쪽:** 8개 단계 버튼 (1~8단계 모두 사용 가능)
- **오른쪽 위:** 수임처 목록 테이블 + 실행 버튼
- **오른쪽 아래:** 진행 상황을 실시간으로 보여주는 로그 화면
- **상단:** 연도/월 선택, 일시정지 버튼
- **상단 툴바:** 1단계 선택 시 "담당자" 이름 입력란 표시, 7/8단계 선택 시 "비밀번호" 입력란 표시

## 6. 1단계: 수임처 목록 조회

WEHAGO에서 담당 수임처 목록을 자동으로 가져옵니다. 다른 모든 단계를 진행하기 전에 먼저 수행해야 합니다.

**절차:**

1. 왼쪽 메뉴에서 **"1 수임처리스트"** 버튼을 클릭합니다
2. 상단 툴바에 **"담당자"** 입력란이 나타납니다
3. (선택) 본인이 담당하는 수임처만 가져오려면 **담당자 이름**을 입력합니다. 비워두면 전체 수임처를 가져옵니다
4. 오른쪽 화면에 **"새로 가져오기"** 버튼이 나타납니다
5. **"새로 가져오기"** 버튼을 클릭합니다
6. Chrome 브라우저가 자동으로 열리며 WEHAGO 로그인 화면이 나타납니다
7. **공동인증서로 직접 로그인**합니다
8. 로그인이 완료되면 수임처 목록이 자동으로 수집됩니다 (담당자 이름을 입력한 경우 해당 수임처만 필터링)
9. 수집이 완료되면 오른쪽 테이블에 수임처 목록이 표시됩니다

> 안내: 수임처 목록은 프로그램에 자동 저장됩니다. 프로그램을 껐다 켜도 목록이 유지됩니다.
> 팁: 담당자 입력란에 이름을 입력하면 WEHAGO 수임처관리에서 해당 담당자의 수임처만 검색하여 가져옵니다.

## 7. 2단계: 국민건강보험 결정내역 다운로드

수임처별로 국민건강보험 결정내역을 PDF, Excel 파일로 자동 다운로드합니다.

**절차:**

1. 화면 상단에서 **조회할 연도와 월**을 선택합니다 (예: 2026년 4월)
2. 왼쪽 메뉴에서 **"2 건강보험"** 버튼을 클릭합니다
3. 화면의 **"전체실행"** 버튼을 클릭합니다
4. Chrome 브라우저가 자동으로 열리며 국민건강보험 EDI 로그인 화면이 나타납니다
5. **공동인증서로 직접 로그인**합니다
6. 로그인이 완료되면 수임처별로 자동 처리가 시작됩니다
7. 오른쪽 테이블에서 각 수임처의 진행 상태를 실시간으로 확인할 수 있습니다
8. 하단 로그 화면에서 상세 진행 내용을 확인할 수 있습니다

**저장 위치:** 바탕화면에 **"국민건강보험_YYYYMM"** 폴더가 자동 생성되며, 그 안에 각 수임처명 폴더별로 PDF와 Excel 파일이 저장됩니다.

## 8. 3단계: 국민연금 결정내역 다운로드

수임처별로 국민연금 결정내역을 PDF, Excel 파일로 자동 다운로드합니다. 진행 방식은 2단계와 동일합니다.

**절차:**

1. 화면 상단에서 **조회할 연도와 월**을 선택합니다
2. 왼쪽 메뉴에서 **"3 국민연금"** 버튼을 클릭합니다
3. 화면의 **"전체실행"** 버튼을 클릭합니다
4. Chrome 브라우저가 자동으로 열리며 국민연금 EDI 로그인 화면이 나타납니다
5. **공동인증서로 직접 로그인**합니다
6. 로그인이 완료되면 수임처별로 자동 처리가 시작됩니다
7. 진행 상황은 오른쪽 테이블과 하단 로그에서 실시간 확인합니다

**저장 위치:** 바탕화면에 **"국민연금_YYYYMM"** 폴더가 자동 생성되며, 그 안에 각 수임처명 폴더별로 PDF와 Excel 파일이 저장됩니다.

## 9. 6단계: 원천이행상황신고서 마감 처리

WEHAGO에서 원천징수이행상황신고서(SWTA0101)의 마감/마감해제를 수임처별로 자동 처리합니다. 2~5단계와 독립적으로 동작하므로 선행 단계 없이도 실행할 수 있습니다.

**절차:**

1. 화면 상단에서 **해당 연도와 월**을 선택합니다
2. 왼쪽 메뉴에서 **"6 원천이행상황신고서"** 버튼을 클릭합니다
3. 화면의 **"전체실행"** 또는 **"선택건 실행"** 버튼을 클릭합니다
4. Chrome 브라우저가 자동으로 열리며 WEHAGO 로그인 화면이 나타납니다
5. **공동인증서로 직접 로그인**합니다
6. 로그인이 완료되면 수임처별로 자동 처리가 시작됩니다:
   - WEHAGO 메인 → 수임처 급여 페이지 진입
   - SWTA0101 메뉴 이동 → 신고유형(매월/반기) 자동 감지
   - 기간 설정 → 조회 → 마감 또는 마감해제 자동 처리
7. 진행 상황은 오른쪽 테이블과 하단 로그에서 실시간 확인합니다

> 안내: 이미 마감된 수임처는 자동으로 감지하여 **마감을 해제** 처리합니다. 미마감 수임처는 마감을 적용합니다. 마감해제 후 버튼이 '마감'으로 전환되지 않으면(예: 확인 모달 미감지) 해당 수임처는 실패로 표시되어 재점검할 수 있습니다.

## 10. 7단계: 원천전자신고 파일 제작

WEHAGO에서 원천징수전자신고(SWER0101) 파일을 수임처별로 자동 제작합니다. 2~6단계와 독립적으로 동작하므로 선행 단계 없이도 실행할 수 있습니다.

**절차:**

1. 화면 상단에서 **해당 연도와 월**을 선택합니다
2. 왼쪽 메뉴에서 **"7 전자신고"** 버튼을 클릭합니다
3. 상단 툴바에 **비밀번호 입력란**이 나타납니다 — 전자신고 파일 비밀번호를 입력합니다 (8~15자리)
4. 화면의 **"전체실행"** 또는 **"선택건 실행"** 버튼을 클릭합니다
5. Chrome 브라우저가 자동으로 열리며 WEHAGO 로그인 화면이 나타납니다
6. **공동인증서로 직접 로그인**합니다
7. 로그인이 완료되면 수임처별로 자동 처리가 시작됩니다:
   - WEHAGO 메인 → 수임처 급여 페이지 진입
   - SWER0101 메뉴 이동 → 지급기간 자동 설정
   - 수임처 선택 → 제작(F4) → 전자신고 파일 제작
   - WehagoNTS 폴더 선택 → 파일 자동 저장
8. 진행 상황은 오른쪽 테이블과 하단 로그에서 실시간 확인합니다

> 안내: 전자신고 파일은 바탕화면의 **"원천전자신고_YYYYMM"** 폴더 안에 수임처별로 정리되어 자동 저장됩니다.

## 11. 8단계: 홈택스 원천세 파일변환신고

7단계에서 생성된 전자신고 파일(`.01`)을 홈택스에 업로드하여 파일검증을 수행합니다. **7단계를 먼저 완료**해야 합니다.

**절차:**

1. 화면 상단에서 **해당 연도와 월**을 선택합니다 (7단계와 동일한 연도/월)
2. 왼쪽 메뉴에서 **"8 홈택스신고"** 버튼을 클릭합니다
3. 상단 툴바에 **비밀번호 입력란**이 나타납니다 — 전자신고 파일 비밀번호를 입력합니다
4. 화면의 **"전체실행"** 또는 **"선택건 실행"** 버튼을 클릭합니다
5. Chrome 브라우저가 자동으로 열리며 홈택스 로그인 화면이 나타납니다
6. **공동인증서로 직접 로그인**합니다
7. 로그인이 완료되면 수임처별로 자동 처리가 시작됩니다:
   - 7단계 결과물(`.01` 파일) 자동 탐색
   - 원천세 신고 > 일반신고 > 파일변환신고 이동
   - 파일 업로드 → 파일검증 수행
8. 진행 상황은 오른쪽 테이블과 하단 로그에서 실시간 확인합니다

> 안내: 파일검증까지만 자동 수행됩니다. 실제 제출은 홈택스 화면에서 직접 진행해 주세요.
> 팁: 홈택스 세션은 20분 주기로 자동 연장됩니다.

## 12. 자주 묻는 질문 (FAQ)

---

**Q: "수임처 조회 실패: no such table: steps" 에러 메시지가 나타납니다**

A: 내부 DB 초기화가 누락된 것입니다. 설치 파일을 다시 실행해 프로그램을 재설치(데이터는 보존됨)하거나, 담당자에게 문의하세요.

---

**Q: Chrome 브라우저가 열리지 않습니다**

A: [공식 사이트](https://www.google.com/chrome/)에서 **Google Chrome(stable)** 이 설치되어 있고, 설치 후 한 번이라도 실행된 상태인지 확인하세요. 네이버 웨일·엣지·브레이브 등 크로미움 기반 브라우저는 인식하지 못합니다.

---

**Q: 로그인 화면에서 프로그램이 멈춰 있는 것 같습니다**

A: 정상입니다. 프로그램은 사용자가 직접 로그인할 때까지 최대 15분간 기다립니다. 공동인증서로 로그인하시면 자동으로 다음 단계로 진행됩니다.

---

**Q: 처리 중간에 에러가 발생해서 멈췄습니다**

A: **"전체실행"** 버튼을 다시 누르면 이미 완료된 수임처는 자동으로 건너뛰고, 실패한 수임처부터 이어서 진행합니다.

---

**Q: 수임처 목록에 누락된 곳이 있습니다**

A: 1단계에서 **"새로 가져오기"** 버튼을 다시 클릭하면 WEHAGO에서 최신 수임처 목록을 다시 가져옵니다.

---

**Q: 프로그램을 중간에 중단하고 싶습니다**

A: 실행 중일 때 **"전체실행"** 버튼이 **"정지"** 로 바뀝니다. 이 버튼을 누르면 안전하게 중단됩니다. 다음에 다시 실행하면 이어서 진행할 수 있습니다.

---

**Q: 프로그램을 일시정지하고 싶습니다**

A: 화면 상단의 **"일시정지"** 버튼을 누르면 잠시 멈춥니다. 다시 누르면(재개) 계속 진행됩니다.

---

**Q: 다운로드된 파일은 어디에서 확인하나요**

A: 바탕화면에 사이트별 폴더가 자동으로 생성됩니다.
- 국민건강보험: 바탕화면 > "국민건강보험_YYYYMM" > 수임처명 폴더
- 국민연금: 바탕화면 > "국민연금_YYYYMM" > 수임처명 폴더
