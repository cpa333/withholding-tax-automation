# 원천징수 자동화 사용자 가이드

## 프로그램 실행

### 로그인

프로그램을 실행하면 로그인 화면이 나타납니다.

1. **이메일**과 **비밀번호**를 입력합니다
2. **로그인** 버튼을 클릭합니다
3. 로그인에 성공하면 자동으로 메인 화면으로 이동합니다

> ℹ️ 계정은 관리자가 사전에 등록합니다. 직접 회원가입할 수 없습니다.
> 계정이 필요한 경우 관리자에게 문의해 주세요.

### 자동 로그인

최초 로그인 이후에는 프로그램을 다시 실행해도 자동으로 로그인됩니다.
별도로 로그인을 입력할 필요가 없습니다.

### 로그아웃

화면 하단 상태바 오른쪽에서 로그아웃할 수 있습니다:

```
┌─────────────────────────────────────────────────────────┐
│  ...                                                     │
├─────────────────────────────────────────────────────────┤
│  줄비                              👤 user@firm.com 로그아웃 │
└─────────────────────────────────────────────────────────┘
```

1. 상태바 우측의 **로그아웃** 버튼을 클릭합니다
2. 로그인 화면이 다시 나타납니다
3. 다른 계정으로 로그인하거나 **종료** 버튼으로 프로그램을 닫을 수 있습니다

> ⚠️ 자동화 작업이 진행 중인 경우 로그아웃 시 확인 메시지가 표시됩니다.

### 세션 만료

장시간 사용 시 인증이 만료될 수 있습니다. 만료된 경우:

1. "인증이 만료되었습니다" 안내가 표시됩니다
2. 다시 이메일과 비밀번호를 입력하여 로그인하세요
3. 기존 작업 상태는 그대로 유지됩니다

## 주요 기능

### 1. 수임처 관리 (Phase 1)

- **새로 가져오기**: WEHAGO에서 수임처 목록을 불러옵니다
- **전체 삭제**: 등록된 수임처를 모두 삭제합니다

### 2. 자동화 실행 (Phase 2~7)

1. 상단에서 **연도**와 **월**을 선택합니다
2. 왼쪽 사이드바에서 실행할 **단계(Phase)**를 선택합니다
3. 수임처 테이블에서 실행할 항목을 선택합니다
4. **선택건 실행** 또는 **전체실행** 버튼을 클릭합니다

### 3. Dry-run 모드

- **dry-run** 체크박스가 선택된 상태에서는 실제 자동화를 수행하지 않고 흐름만 시뮬레이션합니다
- 테스트 목적으로 사용하세요

### 4. 업데이트

- 프로그램이 자동으로 업데이트를 확인합니다
- 새 버전이 있으면 설치 안내가 표시됩니다
- **도움말 → 업데이트 확인**으로 수동 확인도 가능합니다

## 자주 묻는 질문

**Q: 로그인 화면이 나오지 않고 바로 메인 화면이 열립니다.**
A: 이전 로그인 세션이 유지되고 있는 것입니다. 로그아웃하려면 화면 하단 상태바의 "로그아웃" 버튼을 클릭하세요.

**Q: "서버에 연결할 수 없습니다"라는 메시지가 나옵니다.**
A: 인터넷 연결을 확인해 주세요. 인터넷이 잠시 끊긴 경우에도 이전 로그인 후 3일 이내라면 프로그램을 계속 사용할 수 있습니다.

**Q: 인터넷이 안 되는 환경에서도 사용할 수 있나요?**
A: 최초 로그인 시에는 인터넷이 필요합니다. 이후에는 마지막 로그인으로부터 3일 이내라면 오프라인에서도 사용할 수 있습니다.

**Q: 계정을 잊어버렸습니다.**
A: 관리자에게 문의해 주세요. Supabase 대시보드에서 비밀번호를 재설정할 수 있습니다.

**Q: "사용 기간 만료"라는 메시지가 나옵니다.**
A: 베타 사용 기간이 종료되었습니다. 새 버전을 설치해 주세요.
