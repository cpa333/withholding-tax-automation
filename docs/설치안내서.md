# 원천징수 자동화 — 설치 안내 (1분)

> 처음 받는 분을 위한 설치 가이드입니다. 순서대로 따라 하시면 됩니다.
> 막히는 부분은 맨 아래 **문의처**로 연락 주세요.

---

## 1. 설치 파일 다운로드

아래에서 설치 파일을 받습니다.

**다운로드:** [여기에 카페 글 링크 또는 다운로드 URL 기입]

> 💡 **USB나 사내 공유 폴더**로 파일을 직접 받으신 경우 → 아래 **2번(파란 화면)은 뜨지 않습니다**. 바로 **3번**으로 넘어가세요.

---

## 2. 파란 화면이 뜨면 (정상입니다!)

인터넷에서 받은 파일이라 Windows가 한 번 더 확인합니다. **버그나 바이러스가 아니니 안심하세요.**

> 📷 *[이 자리에 SmartScreen 파란 화면 캡처 삽입]*
>
> 화면에 이렇게 뜹니다:
> - 제목: **"Windows에서 PC를 보호했습니다"**
> - 파란 **"실행하지 않음"** 버튼

**이렇게 넘기세요:**

1. 화면 아래 작은 글씨 **"추가 정보"** 클릭
2. 그러면 **"실행"** 버튼(회색)이 나타납니다 → 클릭

> 📷 *[이 자리에 "추가 정보 → 실행" 버튼 캡처 삽입]*

---

## 3. 설치

1. 설치 마법사가 열리면 **[다음]** → **[설치]** → **[완료]** 순서로 클릭
2. 바탕화면에 **"원천징수 자동화"** 아이콘이 생깁니다

> ⚠️ **Google Chrome이 없으면 설치가 안 됩니다.**
> 설치 중 "Chrome이 필요합니다" 창이 뜨면, 먼저 Chrome을 설치하세요.
> Chrome 다운로드: https://www.google.com/chrome/
> Chrome 설치 후 이 설치 파일을 다시 실행하면 됩니다.

---

## 4. 첫 실행 — 로그인

1. 바탕화면 아이콘을 더블클릭해 실행
2. **로그인 창**이 뜹니다
3. **계정(이메일/비밀번호)은 담당자에게 받으세요.**

> 📷 *[이 자리에 로그인 창 캡처 삽입]*
>
> 로그인 창에 **"계정이 없으신가요? 계정 발급은 담당자에게 문의해 주세요."** 안내가 있습니다.

---

## 5. Chrome 자동 연결

로그인 후 작업을 시작하면, **PC에 설치된 Chrome이 자동으로 열려** 프로그램과 연결됩니다. (별도 설치·설정 불필요)

---

## 백신에 차단된 경우 (오류 0x800700E1)

설치/실행 시 *"파일에 바이러스 스캐너가 악의적인 소프트웨어로 간주하여..."* 창이 뜨면 Windows Defender 오탐입니다. **바이러스가 아닙니다.**

**빠른 해결 (도우미 사용):**
1. 배포 파일의 **`add_defender_exclusion.bat`**를 **우클릭 → "관리자 권한으로 실행"**
2. *"[완료]"*가 뜨면 설치 파일을 다시 실행

**수동 해결:**
1. **Windows 보안** → **바이러스 및 위협 방지** → **보호 기록** → 원천징수 자동화 **복원**
2. 같은 화면 **제외 추가 → 폴더**에 아래 두 경로 추가:
   - `%LOCALAPPDATA%\원천징수자동화`
   - `%LOCALAPPDATA%\원천징수자동화-data`

> 상세 화면/캡처는 USER_GUIDE.md "13. 백신(Windows Defender) 오탐 대처" 참고.
