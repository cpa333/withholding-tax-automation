# Tally 복붙용: 회계 AX 후속 과정 우선 대기명단

> 실제 폼 URL: https://tally.so/r/5B5dEb
> 사용법: Tally 새 폼을 만든 뒤 아래 순서대로 붙여넣는다.  
> 선택지가 있는 질문은 먼저 질문 블록을 만든 다음, 첫 번째 선택지 칸에 옵션 목록을 줄바꿈 그대로 붙여넣는다.

## 0. 추천 작업 순서

1. Tally에서 새 폼을 만든다.
2. 제목과 설명문을 먼저 붙여넣는다.
3. 각 질문 위치에서 `/email`, `/check`, `/multiple`, `/long`, `/thank`를 입력해 필드 타입을 만든다.
4. 선택지 목록은 해당 질문의 첫 번째 옵션 칸에 줄바꿈 그대로 붙여넣는다.
5. Q1, Q2, Q5만 Required로 설정한다.
6. Thank-you page를 추가하고 완료 메시지를 붙여넣는다.

## 1. 폼 제목

```text
회계 AX 후속 과정 우선 대기명단
```

## 2. 폼 설명문

```text
오늘은 "이렇게도 할 수 있다"를 보여드렸습니다.
후속 과정은 "내 업무에 실제로 세팅한다"에 가깝습니다.

관심 있는 주제를 선택해 주시면, 대기명단 등록자부터 일정과 커리큘럼을 우선 안내드리겠습니다.
```
