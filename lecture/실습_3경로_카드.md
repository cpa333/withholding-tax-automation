# 실습 진입 3경로 안내 카드 (중간 대휴식 때 배포)

> **오늘 실습은 누구도 낙오하지 않습니다.** 아래 셋 중 본인 상황에 맞는 하나로 따라오세요.
> 설치가 안 되셔도 **웹으로 100% 똑같이** 하실 수 있습니다.

---

### 🟢 경로 A — 웹 (설치 0 · 전원 권장·보장)
1. `claude.ai` 로그인 → 좌측 **Projects** → 새 프로젝트
2. **'커스텀 지침(Custom Instructions)'** 칸을 엽니다
3. 강사가 부르는 대로 **CLAUDE.md 뼈대**(【 】 3칸)를 붙여넣고 저장
4. 같은 세법 질문을 다시 → 내 기준·말투로 답하면 성공 ✅

*→ 대부분 이 경로면 충분합니다. 헷갈리면 손 드세요, 조교가 옆에서 도와드립니다.*

---

### 🔵 경로 B — 노트북 Claude Code (원하는 소수만)
1. 사전 안내 메일의 설치 매뉴얼대로 Claude Code 실행
2. 실습 폴더 `회계실습/` 열기 → 루트에 `CLAUDE.md` 저장
3. 같은 질문 재실행 → 반영 확인 ✅

*→ 안 되면 바로 **경로 A(웹)**로 넘어오세요. 개념은 100% 똑같습니다.*

---

### 🟡 경로 C — 폰만 있어도 (최후·맛보기)
1. 폰 웹 AI(ChatGPT/Claude 앱)에
2. **"역할: ○○ 세무사 / 말투: ~해요체"** 한 줄 라벨을 질문 앞에 붙이기
3. 답 말투가 바뀌는 느낌 확인 ✅
